# BigDataStructures

Реализация классических и продвинутых структур данных и алгоритмов на языке **Go**. Проект создан в рамках университетских лабораторных работ и демонстрирует пошаговое построение основных структур данных, применяемых в области больших данных и компьютерных наук.

## Структуры данных

| Структура | Пакет | Описание |
|-----------|-------|----------|
| **Binary Search Tree** | `bst` | Обобщённое бинарное дерево поиска с операциями вставки, поиска, удаления и трёмя обходами (прямой, симметричный, обратный) |
| **B-Tree** | `btree` | Обобщённое B-дерево с настраиваемой минимальной степенью `t`. Полная реализация вставки и удаления с слиянием/заимствованием у соседних узлов |
| **Bloom Filter** | `bloomfilter` | Вероятностная структура для проверки принадлежности элемента множеству. Использует MurmurHash3, автоматически вычисляет оптимальные параметры |
| **Count-Min Sketch** | `countminsketch` | Вероятестенная структура для оценки частоты элементов в потоке данных. Хеширование через FNV-64a |
| **Trie** | `trie` | Префиксное дерево с фиксированным алфавитом. Поддерживает вставку, поиск, проверку префикса и удаление |
| **Hash Table** | `hashtable` | Обобщённая хеш-таблица с методом цепочек (separate chaining) |
| **Segment Tree** | `segmenttree` | Обобщённое дерево отрезков для запросов на отрезке с обновлением в точке |
| **Fenwick Tree (BIT)** | `fenwicktree` | Обобщённое дерево Фенвика для эффективных префиксных сумм. Работает с произвольной моноидной операцией |
| **Binary Heap** | `heaps` | Обобщённая бинарная куча (макс-куча) на основе массива |
| **Binomial Heap** | `heaps` | Обобщённая биномиальная куча с поддержкой слияния двух куч |

## Алгоритмы

| Алгоритм | Пакет | Описание |
|----------|-------|----------|
| **Euler's Totient φ(n)** | `euler` | Три реализации: перебором, через разложение на простые, через ДПФ |
| **DFT / Inverse DFT** | `euler` | Дискретное преобразование Фурье через умножение на матрицу Вандермонда |
| **FFT (Cooley-Tukey)** | `euler` | Быстрое преобразование Фурье (radix-2) с fallback для размеров, не являющихся степенью двойки |
| **Primitive Roots in Z_n** | `euler` | Поиск примитивных корней по модулю n, проверка существования |

## Запуск

```bash
# Клонирование
git clone https://github.com/code-stepan/BigDataStructures.git
cd BigDataStructures

# Запуск интерактивной демонстрации
go run main.go
```

После запуска отобразится меню с 12 пунктами — каждый запускает демо соответствующей структуры данных или алгоритма.

## Требования

- Go 1.21+ (используются дженерики)

## Зависимости

- [`github.com/spaolacci/murmur3`](https://github.com/spaolacci/murmur3) — реализация MurmurHash3 для Bloom Filter
